# Milestone 4: 混合证据展示优化 (Hybrid Evidence UI & Interaction)

## 目标 (Goal)
优化 Hybrid Evidence（混合检索证据）的展示方式，提升用户阅读体验，支持按需查看详细引用。

## 需求详情 (Requirements)

### 1. 交互优化 (Interaction)
- **展开/折叠控制**:
    - 在分数的右侧增加一个 **展开/折叠按钮** (chevron icon)。
    - **点击交互**: 点击整个卡片区域，或者点击展开按钮，均可切换展开/折叠状态。
    - **默认状态**: 默认折叠 (Collapsed)，只显示标题、分数和简短摘要（如果现有）。

### 2. 详细展示 (Expanded View)
- 当卡片处于 **展开态** 时，需要显示更详细的检索信息：
    - **引用章节 (Section)**: 显示该条证据所属的章节标题。
    - **原文片段 (Quote)**: 显示检索到的具体原文文本。

### 3. 本文样式 (Styling)
- **高亮反馈**:
    - 当卡片展开时，**卡片边框** 应变色（例如高亮色），以区分当前阅读焦点。
- **引用样式**:
    - 引用文字 (Quote) 需要有明显的视觉区分。
    - 建议样式：左侧带有竖线 (border-left) 并添加轻微的背景色 (background color)，使其看起来像是一个 Quote Block。
