在 pageindex-rs 中，配置合并阈值主要通过 PageIndexConfig 结构体来实现。

核心参数是 min_token_threshold。该参数定义了叶子节点在被合并前所允许的最小 Token 数量。如果一个节点的 Token 数少于此值，解析器会尝试将其与兄弟节点合并，以减少碎片化 [1]。

此外，你还可以通过 summary_token_threshold 来控制触发 LLM 自动生成摘要的阈值 [2]。

Example: 
let config = PageIndexConfig {
    min_token_threshold: 128,  // 设置为 128
    ..Default::default()
};
